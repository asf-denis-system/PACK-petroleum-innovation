# [UP.M.014] Три фабрики (проблем, решений, фабрик)

## Определение

Фреймворк из трёх уровней организации инновационной работы. Каждая фабрика — самостоятельный цикл с входами и выходами.

## Три уровня

### Фабрика проблем
Характеризация → измерение → индикаторизация → проблематизация → спецификация сравнения.

**Выход:** приоритизированный список проблем с измеримыми индикаторами.

### Фабрика решений
Генерация вариантов → сравнение → выбор → реализация → замер эффекта → пакет свидетельств.

**Выход:** реализованные решения с доказательной базой.

### Фабрика фабрик
Оргразвитие (DevOps/SRE-аналог): улучшение самих процессов генерации проблем и решений.

**Выход:** улучшенные процессы, регламенты, инструменты.

## Ключевые различения

- Фабрика проблем ≠ фабрика решений (нельзя смешивать характеризацию с генерацией)
- Портфель вариантов > ставка на один вариант (ошибка недоисследованного пространства решений)
- Проигравший вариант ≠ ошибка (различение D.001 из PACK-personal)
